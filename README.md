# Documentation du projet

## Description :

Le jeu as été développé en python avec la bibliothèque pygame, il n'a pas de fonctionalité complexe, on incarne un personnage qui doit éviter les obstacles en sautant, et possède 3 vies maximum.
N'hésitez pas à déposer une étoile si vous apréciez le projet.

Les touches sont :
- Espace : pour sauter
- P ou ESC pour mettre en pause

Pour voir les hitbox, il faut mettre le booléen hitbox_activee à true en haut de main.py

## Lancer le jeu :

Il faut avoir python et pygame installés, puis lancer le fichier main.py
```bash
python -m ensurepip --upgrade

pip install pygame
python main.py
```
Reste plus qu'a s'amuser !
